# WebdriverTest

文件夹说明：
Data-driven:存储数据的目录                             <br>
Model：数据层的包，对数据的处理，例如对excel，xml,mysql等的处理方法，供TestCase中的test_baidu.py调用                             <br>
       config.py为配置文件，涉及到配置的写到次，例如连接数据库的地址/端口等配置信息                             <br>
TestCase:测试层的包                             <br>
Page:对象层的包                              <br>
Report:测试报告目录                              <br>
allTests:执行所有测试模块的文件                              <br>
